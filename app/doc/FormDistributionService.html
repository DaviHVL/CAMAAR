    class FormDistributionService

    Service Object responsável por clonar a estrutura de um Template e distribuí-lo para múltiplas turmas na forma de formulários independentes.

    Encapsula a lógica de “Deep Copy” (cópia profunda) de Template -> Formulario, garantindo que cada turma tenha sua própria instância de avaliação.

def call
  ActiveRecord::Base.transaction do
    @turma_ids.each do |tid|
      create_form_copy_for_class(tid)
    end
  end
end

    Executa o processo de distribuição dentro de uma transação de banco de dados. Garante atomicidade: ou todos os formulários são criados, ou nenhum é.

    Args: Nenhum

    Retorno: O resultado da execução do bloco de transação (geralmente a lista de IDs processada).

    Efeitos Colaterais:

    - Cria registros nas tabelas: Formulario, FormularioTurma, QuestaoFormulario e OpcaoFormulario.
