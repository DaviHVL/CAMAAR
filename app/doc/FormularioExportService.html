    class FormularioExportService

    Service Object responsável pela exportação dos dados de um formulário para o formato CSV.

    Encapsula a lógica de formatação de colunas, cabeçalhos dinâmicos (baseados nas perguntas) e a consolidação das respostas dos alunos em linhas.

def initialize(formulario)
  @formulario = formulario
  @questoes = @formulario.questao_formularios.order(:id)
end

    Inicializa o serviço preparando os dados necessários. Carrega as questões ordenadas por ID para garantir que as colunas do CSV sigam sempre a mesma ordem lógica.

    Args:

def call
  CSV.generate(headers: true) do |csv|
    csv << headers

    responses_scope.each do |resposta_geral|
      csv << build_row(resposta_geral)
    end
  end
end

    Executa a geração do conteúdo CSV. Cria a estrutura do arquivo, adiciona o cabeçalho e itera sobre todas as respostas.

    Args: Nenhum

    Retorno:

    - String: O conteúdo textual completo do CSV gerado.

    Efeitos Colaterais:

    - Dispara consultas ao banco de dados (via responses_scope).
- Não altera dados no banco.
