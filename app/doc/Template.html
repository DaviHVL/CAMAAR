    class Template

    Representa o “molde” ou estrutura base de um questionário de avaliação. É criado por um administrador e contém o conjunto de perguntas (QuestaoTemplate) que serão futuramente copiadas para criar os Formulários reais enviados às turmas.

def build_initial_structure
  questao = questao_templates.build
  questao.opcao_templates.build
end

    Constrói a estrutura inicial de objetos aninhados necessária para renderizar o formulário de criação “nested” na view.

    Inicializa uma QuestaoTemplate vazia e, dentro dela, uma OpcaoTemplate vazia, garantindo que o usuário veja pelo menos um campo de pergunta ao abrir a tela “Novo”.

    Args: Nenhum

    Retorno:

    - OpcaoTemplate: O último objeto instanciado na cadeia (instância vazia).

    Efeitos Colaterais:

    - Altera o estado do objeto em memória adicionando associações vazias.
- Não salva nada no banco de dados.
